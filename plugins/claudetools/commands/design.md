---
description: "Brainstorm approaches and write an implementation plan."
argument-hint: "[feature or task description]"
---

This is a workflow command. Follow this sequence:
1. First invoke the `claudetools:brainstorm` skill with the user's description
2. After design approval, invoke the `claudetools:plan` skill to create the implementation plan
3. Tell the user: "Design and plan complete. **Next: /build** to implement the plan."
