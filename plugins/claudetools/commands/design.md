description: "Brainstorm approaches and write an implementation plan. Delegates to superpowers brainstorming and writing-plans skills."
argument-hint: "[feature or task description]"
---

This is a workflow command. Follow this sequence:
1. First invoke `superpowers:brainstorming` with the user's description
2. After design approval, invoke `superpowers:writing-plans` to create the implementation plan
3. Tell the user: "Design and plan complete. **Next: /build** to implement the plan."
