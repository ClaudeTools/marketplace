## What claudetools fixes

<CardGrid stagger>
  <Card title="Claude breaks your build" icon="shield">
    Ever had Claude run `rm -rf` or push directly to main? Safety hooks intercept destructive commands before they execute — and explain what went wrong.
  </Card>
  <Card title="Debugging goes in circles" icon="warning">
    Claude guesses, reverts, guesses again. The debugger skill enforces a 6-step protocol: reproduce first, fix last. No more patch-and-pray.
  </Card>
  <Card title="Unfamiliar codebases take forever" icon="magnifier">
    Reading file by file to understand a project wastes tokens and time. Codebase Pilot indexes your repo with tree-sitter — find symbols, trace imports, map architecture in seconds.
  </Card>
  <Card title="Long sessions lose context" icon="puzzle">
    Claude forgets what it decided three messages ago. The memory system carries decisions, preferences, and project state across every session automatically.
  </Card>
</CardGrid>

## claudetools vs manual Claude Code

| Without claudetools | With claudetools |
|---------------------|------------------|
| Claude edits files it hasn't read | `pre-edit-gate` hook blocks blind edits until the file is read |
| `rm -rf` and force-pushes run unchecked | Safety hooks intercept destructive commands before they execute |
| Debugging is patch-and-pray — guess, revert, repeat | `/debugger` enforces REPRODUCE → HYPOTHESIZE → VERIFY → FIX |
| Unfamiliar codebases require reading dozens of files | Codebase Pilot indexes the repo — find symbols, trace imports in seconds |
| Sessions forget context from earlier in the conversation | Memory system persists decisions, preferences, and project state across sessions |
| Multi-agent work produces conflicting edits | Agent mesh coordinates file locks and shared decisions across teammates |
| You manually track what changed and what's pending | `/task-manager` creates persistent tasks with cross-session handoffs |
| Code reviews happen late in the PR cycle | `/code-review` runs a 4-pass structured review on any diff, locally |
