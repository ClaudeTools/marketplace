# Skills

7 intelligent workflows that Claude invokes automatically when your task matches, or on demand via `/skill-name`.

| Skill | Trigger |
|-------|---------|
| [Exploring Codebase](/reference/skills/exploring-codebase/) | Understanding code, finding definitions, tracing dependencies |
| [Investigating Bugs](/reference/skills/investigating-bugs/) | Debugging errors, fixing failures, diagnosing issues |
| [Improving Prompts](/reference/skills/improving-prompts/) | Structuring prompts, task decomposition |
| [Designing Interfaces](/reference/skills/designing-interfaces/) | Building UI, design systems, responsive layouts |
| [Managing Tasks](/reference/skills/managing-tasks/) | Task persistence, cross-session continuity, handoff |
| [Evaluating Safety](/reference/skills/evaluating-safety/) | Training scenarios, safety testing, model evaluation |
| [Improving Plugin](/reference/skills/improving-plugin/) | Self-improvement loop with before/after measurement |
